# web-nav 全面改建设计文档

## 1. 概述

对个人导航门户 web-nav 进行全面改版，以**极简效率风**为主视觉方向，辅以毛玻璃质感点缀，融合仪表盘功能 Widget。

## 2. 技术架构

- **技术栈**：纯静态 HTML / CSS / JavaScript（保持不变）
- **依赖移除**：去除未使用的 Bootstrap、jQuery、Popper.js、Font Awesome 依赖，改为原生实现
- **单文件架构**：保持单个 index.html 结构，所有样式和脚本内联
- **数据源**：继续使用 links.json 作为链接数据来源
- **构建流程**：无构建步骤，保持零依赖部署

### 3.2 平板端布局（768-1023px）

- 左侧面板默认折叠，通过汉堡菜单或侧滑手势展开
- 分类导航变为可横向滚动的标签栏

- 底部固定导航栏（提供面板入口 + 回到顶部）
- 卡片网格调整为 1-2 列

## 4. 视觉设计

### 4.1 配色方案

**已确认：高级灰（D）**

**极简灰色系配色：**
- 主体背景：`#f8fafc`（浅灰白）
- 卡片背景：`#ffffff`（纯白）
- 品牌色（点缀）：`#6b7280`（中灰）→ hover 渐变为 `#4b5563`
- 主文字：`#111827`（近黑）
- 辅助文字：`#6b7280`（中灰）
- 边框：`#e5e7eb`（浅灰）
- 强调色：可使用一个低饱和度彩色（如 `#3b82f6` 淡蓝）作为链接 hover 的微点缀

### 4.2 CSS 变量体系

```css
:root {
  /* 品牌色 - 极简高级灰 */
  --primary: #6b7280;
  --primary-dark: #4b5563;
  --primary-light: #9ca3af;

  /* 背景 */
  --bg-page: #f8fafc;
  --bg-card: #ffffff;
  --bg-surface: #f1f5f9;
  --bg-glass: rgba(255, 255, 255, 0.7);

  /* 文字 */
  --text-primary: #111827;
  --text-secondary: #6b7280;
  --text-muted: #9ca3af;

  /* 边框 */
  --border: #e5e7eb;
  --border-glass: rgba(255, 255, 255, 0.8);

  /* 阴影 */
  --shadow-sm: 0 1px 2px rgba(0,0,0,0.05);
  --shadow-md: 0 4px 6px rgba(0,0,0,0.07);
  --shadow-lg: 0 10px 25px rgba(0,0,0,0.1);

  /* 圆角 */
  --radius-sm: 8px;
  --radius-md: 12px;
  --radius-lg: 16px;

  /* 毛玻璃 */
  --glass-blur: 16px;

  /* 强调色（链接 hover 微点缀） */
  --accent: #3b82f6;
}
```

### 4.3 排版

- 字体：`-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Noto Sans SC', sans-serif`
- 字号比例：12 / 14 / 16 / 18 / 24 / 32 px
- 行高：1.5（正文）、1.3（标题）
- 字重：400（正文）、500（中等）、600（半粗）、700（粗体）

### 4.4 毛玻璃定位

毛玻璃效果**仅用于以下特定区域**：
- 顶部栏背景
- 左侧面板容器
- 时钟 Widget
- 登录/设置弹窗（如有）

链接卡片**不使用**毛玻璃效果，保持纯色简洁，避免性能开销和视觉干扰。

## 5. 组件设计

### 5.1 顶部栏

| 元素 | 说明 |
|------|------|
| Logo | "个人导航"，简约文字 logo |
| 搜索框 | 圆角输入框，引擎选择按钮嵌入左侧，搜索按钮右侧 |
| 引擎选择 | 下拉菜单或横向按钮组，支持：百度/Google/必应/搜狗/DuckDuckGo/本站搜索 |
| 操作按钮 | 壁纸切换按钮、主题切换（明/暗）按钮 |

### 5.2 左侧面板

固定宽度约 260-280px，内容：

| Widget | 说明 |
|--------|------|
| ⏰ 时钟 | Canvas 表盘时钟，经典指针样式，下方显示日期 |
| 🌤 天气 | 显示当前城市温度、天气图标（通过免费 API 或手动输入） |
| 📅 日历 | 紧凑月视图 + 农历日期 + 法定假日标注（红色标记） |
| 📌 快捷书签 | 用户自定义常用链接（编辑/删除/添加），localStorage 存储 |

### 5.3 分类导航

- 横向按钮组，支持水平滚动（触摸滑动 + 鼠标滚轮）
- 每个按钮：[图标] + 分类名称
- 当前选中态：品牌色填充
- first item 始终为"全部"

### 5.4 链接卡片

| 状态 | 样式 |
|------|------|
| 默认 | 白底、浅阴影、圆角 10px |
| Hover | translateY(-2px)、阴影加深至 --shadow-md |
| 布局 | 居左排列：图标 | 名称 + 简介 |
| 图标 | 域名对应 favicon，加载失败显示 emoji 兜底 |

默认显示前 16 个链接，点击"查看全部"展开当前分类全部链接（筛选模式）。

### 5.5 时钟组件

**已确认：Canvas 表盘时钟**

从现有 time.html 的 Canvas 表盘时钟改造：
- 复用 Canvas 绘制逻辑，优化配色和样式
- 白色底色 + 灰色刻度/指针，秒针用品牌色点缀
- 简约现代指针风格（时分秒三针）
- 适配左侧面板空间（表盘直径约 180-200px）
- 表盘下方显示数字日期
- 暗色模式下表盘底色和指针颜色自动适配

### 5.6 日历组件

在现有日历基础上增强：

**现有功能保留：**
- 月视图网格
- 农历日期显示
- 宜忌（老黄历）
- 节假日高亮

**新增功能：**
- 法定假日列表标注（显示放假天数、调休信息）
- 今日快捷定位按钮（已有，保留）
- 更清晰的假日/调休视觉区分
- 精简紧凑设计，适应左侧面板空间

### 5.7 壁纸切换

- 默认：浅灰纯色背景
- 提供 4-6 套预设壁纸选项（渐变色/图案/风景）
- 支持上传自定义图片（FileReader API）
- 通过 CSS 变量 `--bg-image` 动态切换
- 选择持久化到 localStorage
- 暗色模式下壁纸自动调暗（叠加半透明遮罩）

## 6. 数据流

```
links.json → fetch → allData (内存数组)
                         ↓
               renderCategoryNav() → 分类标签
                         ↓
               filterAndRender() → 过滤 + 搜索 → render() → 卡片HTML

localStorage 用于持久化：
  - searchEngine (搜索引号偏好)
  - wallpaper (壁纸选择)
  - quickLinks (快捷书签)
  - themeMode (明/暗模式)
```

## 7. 交互增强

| 交互 | 说明 |
|------|------|
| 键盘快捷键 | `/` 聚焦搜索、`Esc` 清除搜索、`Ctrl+M` 切换明暗 |

| 点击面板外关闭 | 日历面板、设置弹窗点击外部自动关闭 |

## 8. 响应式断点

| 断点 | 布局 |
|------|------|
| ≥ 1200px | 宽屏优化，卡片最大宽度控制 |
| ≥ 1024px | 标准双栏布局 |
| 768-1023px | 面板折叠，侧边滑出 |
| < 768px | 单栏堆叠，底部导航 |
| < 480px | 小屏优化，字号/间距微调 |

## 9. 暗色模式

- 使用 `@media (prefers-color-scheme: dark)` 自动适配
- 同时提供手动切换按钮，覆盖系统设置
- 暗色模式 CSS 变量覆盖：

```css
[data-theme="dark"] {
  --bg-page: #0f172a;
  --bg-card: #1e293b;
  --bg-surface: #1e293b;
  --bg-glass: rgba(30, 41, 59, 0.8);
  --text-primary: #f1f5f9;
  --text-secondary: #94a3b8;
  --text-muted: #64748b;
  --border: #334155;
  --border-glass: rgba(51, 65, 85, 0.8);
}
```

## 11. 性能目标

- 首屏加载时间（无缓存）< 1.5s
- 移除冗余依赖后，总传输体积减少 ~300KB
- 无外部请求（除 links.json 和 favicon 图片外）
- 滚动/交互帧率保持 60fps
